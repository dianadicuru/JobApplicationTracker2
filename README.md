# Job Application Tracker

This project is a Flask and MySQL web application for tracking job applications.

## Features
- CRUD for companies
- CRUD for jobs
- CRUD for skills
- CRUD for applications
- Job Match feature based on matching user skills with required job skills

## Setup
1. Clone the repository
2. Create a virtual environment
3. Install dependencies with `pip install -r requirements.txt`
4. Create a `.env` file with your database credentials
5. Run `schema.sql` in MySQL Workbench
6. Start the app with `python app.py`
