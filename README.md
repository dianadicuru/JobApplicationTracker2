# Job Application Tracker

The Job Application Tracker is a web application that helps users organize and manage their job search. The system allows users to track companies, job postings, applications, and contacts, and provides a job-matching feature that evaluates how well a candidate’s skills align with a job’s requirements.

## Features
- Add, edit, and delete companies, jobs, applications, and contacts
Track application status, resume versions, and interview data
Store structured job requirements using JSON
- Add, edit, and delete companies, jobs, applications, and contacts, track application status, resume versions, and interview data, store structured job requirements using JSON
- Compare candidate skills against job requirements, calculate a match percentage based on overlapping skills, display matched and missing skills

## Setup
1. Clone the repository
2. Create a virtual environment
3. Install dependencies with `pip install -r requirements.txt`
4. Create a `.env` file with your database credentials
5. Run `schema.sql` in MySQL Workbench
6. Start the app with `python app.py`
